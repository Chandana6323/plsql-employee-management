# Employee Management System (Oracle SQL & PL/SQL)

## Overview
This project demonstrates a beginner-friendly Employee Management System built using Oracle SQL and PL/SQL.  
It focuses on core database concepts commonly used in real-world applications.

## Features
- Employee table design
- Stored procedure to add employee records
- PL/SQL function to retrieve employee salary
- Trigger to validate salary and maintain data integrity
- Exception handling for runtime error scenarios

## Technologies Used
- Oracle Database 21c XE
- SQL Developer
- SQL & PL/SQL

## Purpose
This project was created as part of hands-on practice to strengthen PL/SQL fundamentals for entry-level software engineering roles.
